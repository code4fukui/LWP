# LWP

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A library of the LEGO Wireless Protocol (LWP) in JavaScript and WebBluetooth.

## Demo

- [LWP debugger](https://code4fukui.github.io/LWP/debugger.html)
- [Colorpicker by BLE](https://code4fukui.github.io/LWP/colorpicker.html)

## Reference

- [LEGO Wireless Protocol 3.0.00 Doc v3.0.00 r17 documentation](https://lego.github.io/lego-ble-wireless-protocol-docs/)

## Requirements

Requires a Bluetooth-enabled browser that supports the Web Bluetooth API.

## Usage

The library can be used to interact with LEGO Bluetooth devices that support the LWP protocol. It provides functions for connecting to devices, reading and writing data, and handling events.
